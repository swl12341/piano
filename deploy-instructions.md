# GitHub Pages 部署说明

## 启用步骤

1. 访问：https://github.com/swl12341/piano/settings/pages
2. 在 "Source" 下选择 "Deploy from a branch"
3. 选择 "main" 分支
4. 选择 "/ (root)" 文件夹
5. 点击 "Save"

## 访问链接

部署完成后，您的项目将在以下地址可用：
**https://swl12341.github.io/piano**
